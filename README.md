# linux_hardening_lab
This document a hands-on Linux Security hardening lab focused on basic defensive security practices Topics covered include user and permission management, service control, secure SSH configuration, firewall setup, logging, and basic security checks.

The goal of this lab is to demonstrate fundamental knowledge of Linux system administration and cybersecurity principles by securing a fresh Linux system step by step.
This lab is intended for educational purposes and reflects best practices commonly used in real-world environments.
